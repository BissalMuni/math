# LLM에 쓰이는 수학 개념 총정리

## 1. 선형대수 (Linear Algebra)

### 벡터 (Vector)

- **LLM 활용**: 단어를 고차원 벡터로 표현 (Word Embedding). GPT-3는 12,288차원 벡터 사용
- **교육과정**: 고등학교 기하

### 행렬 (Matrix)

- **LLM 활용**: 모든 레이어의 가중치(Weight)가 행렬로 저장. 입력과 가중치의 행렬 곱으로 순전파 수행
- **교육과정**: 대학교 선형대수학
- **공식**: `Y = WX + b` (선형 변환)

### 행렬 곱셈 (Matrix Multiplication)

- **LLM 활용**: Transformer의 핵심 연산. Q, K, V 행렬의 곱으로 Attention Score 계산. LLM 추론 비용의 대부분 차지
- **교육과정**: 대학교 선형대수학
- **공식**: `(AB)ᵢⱼ = Σₖ Aᵢₖ Bₖⱼ`

### 전치 행렬 (Transpose)

- **LLM 활용**: Attention에서 Q·Kᵀ 계산 시 Key 행렬을 전치
- **교육과정**: 대학교 선형대수학
- **공식**: `(Aᵀ)ᵢⱼ = Aⱼᵢ`

### 내적 (Dot Product)

- **LLM 활용**: Scaled Dot-Product Attention의 핵심. Query와 Key의 내적으로 토큰 간 유사도 측정
- **교육과정**: 고등학교 기하
- **공식**: `a · b = Σ aᵢbᵢ = |a||b|cosθ`

### 텐서 (Tensor)

- **LLM 활용**: 배치 처리 시 3차원 이상의 다차원 배열. (batch, sequence_length, hidden_dim) 형태
- **교육과정**: 대학교 선형대수학/텐서해석

### 고유값/고유벡터 (Eigenvalue/Eigenvector)

- **LLM 활용**: 가중치 행렬의 안정성 분석, PCA 기반 차원 축소, 모델 압축
- **교육과정**: 대학교 선형대수학
- **공식**: `Av = λv` (A: 행렬, v: 고유벡터, λ: 고유값)

### 특이값 분해 (SVD)

- **LLM 활용**: LoRA(Low-Rank Adaptation)의 이론적 기반. 가중치 행렬의 저랭크 근사
- **교육과정**: 대학교 선형대수학
- **공식**: `A = UΣVᵀ`

### 코사인 유사도 (Cosine Similarity)

- **LLM 활용**: 임베딩 벡터 간 의미적 유사도 측정. RAG에서 문서-쿼리 매칭
- **교육과정**: 대학교 선형대수학
- **공식**: `cos(θ) = (a · b) / (|a| × |b|)`

### 노름 (Norm)

- **LLM 활용**: Layer Normalization에서 벡터의 크기 정규화. Gradient Clipping에 사용
- **교육과정**: 대학교 선형대수학
- **공식**: `||x||₂ = √(Σ xᵢ²)` (L2 노름)

## 2. 미적분 (Calculus)

### 편미분 (Partial Derivative)

- **LLM 활용**: 역전파(Backpropagation)의 핵심. 손실 함수를 각 가중치에 대해 편미분하여 그래디언트 계산
- **교육과정**: 대학교 다변수 미적분학
- **공식**: `∂f/∂xᵢ` (다른 변수를 고정하고 xᵢ에 대해 미분)

### 연쇄 법칙 (Chain Rule)

- **LLM 활용**: 역전파 알고리즘의 수학적 기반. 합성 함수의 미분을 레이어별로 전파
- **교육과정**: 고등학교 미적분Ⅱ / 대학교
- **공식**: `df/dx = (df/dg) × (dg/dx)` → 다변수: `∂L/∂w = (∂L/∂y)(∂y/∂w)`

### 그래디언트 (Gradient)

- **LLM 활용**: 모든 파라미터에 대한 손실 함수의 편미분 벡터. 파라미터 업데이트 방향 결정
- **교육과정**: 대학교 다변수 미적분학
- **공식**: `∇f = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)`, 업데이트: `θ ← θ - η∇L`

### 지수 함수 (Exponential Function)

- **LLM 활용**: Softmax 함수의 핵심 구성요소. 확률 분포 생성
- **교육과정**: 고등학교 대수
- **공식**: Softmax: `softmax(xᵢ) = eˣⁱ / Σⱼ eˣʲ`

### 로그 함수 (Logarithm)

- **LLM 활용**: Cross-Entropy Loss에서 log 확률 계산. Log-softmax로 수치 안정성 확보
- **교육과정**: 고등학교 대수
- **공식**: `L = -Σ y · log(ŷ)`

### 테일러 급수 (Taylor Series)

- **LLM 활용**: GELU 등 활성화 함수의 근사. 최적화 알고리즘의 이론적 분석
- **교육과정**: 대학교 미적분학
- **공식**: `f(x) = Σₙ f⁽ⁿ⁾(a)/n! × (x-a)ⁿ`

## 3. 확률과 통계 (Probability & Statistics)

### 조건부 확률 (Conditional Probability)

- **LLM 활용**: LLM의 근본 원리. 이전 토큰들이 주어졌을 때 다음 토큰의 확률 모델링
- **교육과정**: 고등학교 확률과 통계
- **공식**: `P(A|B) = P(A∩B) / P(B)`, LLM: `P(xₜ | x₁,...,xₜ₋₁)`

### 결합 확률 (Joint Probability)

- **LLM 활용**: 전체 문장의 확률을 조건부 확률의 곱으로 분해
- **교육과정**: 고등학교 확률과 통계
- **공식**: `P(x₁,...,xₙ) = ∏ P(xₜ | x₁,...,xₜ₋₁)`

### 베이즈 정리 (Bayes' Theorem)

- **LLM 활용**: RLHF에서 보상 모델의 이론적 기반. 베이지안 관점에서의 모델 학습 해석
- **교육과정**: 고등학교 확률과 통계 / 대학교
- **공식**: `P(A|B) = P(B|A)P(A) / P(B)`

### 기대값 (Expected Value)

- **LLM 활용**: 손실 함수가 기대 손실의 추정. RLHF에서 보상의 기대값 최대화
- **교육과정**: 고등학교 확률과 통계
- **공식**: `E[X] = Σ xᵢP(xᵢ)`

### 분산/표준편차 (Variance/Standard Deviation)

- **LLM 활용**: Layer Normalization에서 출력의 분산을 1로 정규화
- **교육과정**: 고등학교 확률과 통계
- **공식**: `Var(X) = E[(X - μ)²]`, `σ = √Var(X)`

### 정규 분포 (Normal Distribution)

- **LLM 활용**: 가중치 초기화 (Xavier, He). GELU 활성화 함수가 가우시안 CDF 기반
- **교육과정**: 대학교 확률과 통계
- **공식**: `f(x) = (1/σ√2π) × e^(-(x-μ)²/2σ²)`

### 최대 우도 추정 (MLE)

- **LLM 활용**: 사전 학습의 목적 함수. 다음 토큰 예측의 로그 우도 최대화 = Cross-Entropy Loss 최소화와 동치
- **교육과정**: 대학교 수리통계학
- **공식**: `θ̂ = argmax_θ Σ log P(xₜ | x₁,...,xₜ₋₁; θ)`

### 샘플링 (Sampling)

- **LLM 활용**: 텍스트 생성 시 확률 분포에서 다음 토큰 샘플링. Temperature로 분포 날카로움 제어
- **교육과정**: 대학교 확률과 통계
- **공식**: Temperature: `P(xᵢ) = eˣⁱ/ᵀ / Σⱼ eˣʲ/ᵀ` (T↓ → 결정적, T↑ → 무작위)

## 4. 정보이론 (Information Theory)

### 엔트로피 (Entropy)

- **LLM 활용**: 확률 분포의 불확실성 측정. 생성 텍스트의 다양성 평가
- **교육과정**: 대학교 정보이론
- **공식**: `H(X) = -Σ P(x) log P(x)`

### 교차 엔트로피 (Cross-Entropy)

- **LLM 활용**: **LLM의 주요 손실 함수.** 모델 예측 분포와 실제 분포 간의 차이 측정
- **교육과정**: 대학교 정보이론
- **공식**: `H(p, q) = -Σ p(x) log q(x)`

### KL 발산 (KL Divergence)

- **LLM 활용**: RLHF에서 미세조정 모델이 원래 모델에서 벗어나지 않도록 KL 페널티 적용. 지식 증류
- **교육과정**: 대학교 정보이론
- **공식**: `D_KL(P||Q) = Σ P(x) log(P(x)/Q(x))`

### 퍼플렉시티 (Perplexity)

- **LLM 활용**: LLM 성능의 핵심 평가 지표. 모델이 다음 토큰을 얼마나 잘 예측하는지 측정
- **교육과정**: 대학교 정보이론/NLP
- **공식**: `PPL = e^H` 또는 `PPL = 2^H` (H = cross-entropy)

### 상호 정보량 (Mutual Information)

- **LLM 활용**: 토큰 간 의존성 분석. 모델이 학습한 표현의 품질 평가
- **교육과정**: 대학교 정보이론
- **공식**: `I(X;Y) = H(X) - H(X|Y) = Σ P(x,y) log(P(x,y) / P(x)P(y))`

## 5. 최적화 (Optimization)

### 경사 하강법 (Gradient Descent)

- **LLM 활용**: 모든 딥러닝 학습의 기본. 손실 함수를 줄이는 방향으로 파라미터 반복 업데이트
- **교육과정**: 대학교 최적화이론
- **공식**: `θₜ₊₁ = θₜ - η∇L(θₜ)` (η: 학습률)

### 확률적 경사 하강법 (SGD)

- **LLM 활용**: 전체 데이터 대신 미니배치로 그래디언트 추정. 계산 효율성 + 정규화 효과
- **교육과정**: 대학교 최적화이론
- **공식**: `θₜ₊₁ = θₜ - η∇L_batch(θₜ)`

### Adam 옵티마이저 (Adam/AdamW)

- **LLM 활용**: LLM 학습의 사실상 표준. 1차/2차 모멘트의 지수 이동 평균으로 적응적 학습률
- **교육과정**: 대학교 최적화이론
- **공식**:
  - `mₜ = β₁mₜ₋₁ + (1-β₁)gₜ` (1차 모멘트)
  - `vₜ = β₂vₜ₋₁ + (1-β₂)gₜ²` (2차 모멘트)
  - `θₜ₊₁ = θₜ - η × m̂ₜ / (√v̂ₜ + ε)`

### 학습률 스케줄링 (Learning Rate Scheduling)

- **LLM 활용**: Warmup + Cosine Decay가 표준. 초기에 학습률을 서서히 올리고 코사인 형태로 감소
- **교육과정**: 대학교 최적화이론
- **공식**: Cosine: `ηₜ = η_min + ½(η_max - η_min)(1 + cos(πt/T))`

### 정규화 (Regularization)

- **LLM 활용**: Weight Decay, Dropout, Label Smoothing. 과적합 방지
- **교육과정**: 대학교 통계학/기계학습
- **공식**: L2 정규화: `L_total = L + λΣwᵢ²`

## 6. 수치해석 (Numerical Methods)

### 부동소수점 산술 (Floating Point Arithmetic)

- **LLM 활용**: FP32, FP16, BF16, INT8, FP4 등 양자화 기법 (GPTQ, AWQ, GGUF)
- **교육과정**: 대학교 수치해석

### 수치 안정성 (Numerical Stability)

- **LLM 활용**: Softmax의 오버플로 방지 (max를 빼는 트릭), LogSumExp 트릭
- **교육과정**: 대학교 수치해석
- **공식**: 안정 softmax: `softmax(xᵢ) = e^(xᵢ - max(x)) / Σⱼ e^(xⱼ - max(x))`

### 행렬 분해 / LoRA (Low-Rank Adaptation)

- **LLM 활용**: 가중치 업데이트를 저랭크 행렬 2개의 곱으로 근사. 파라미터 수를 극적으로 절감
- **교육과정**: 대학교 수치해석/선형대수
- **공식**: `W' = W + ΔW = W + BA` (B: d×r, A: r×d, r << d)

## 7. 기타 수학

### 삼각함수 (Trigonometric Functions)

- **LLM 활용**: Positional Encoding에서 sin/cos으로 토큰 위치 인코딩. RoPE에서 회전 행렬
- **교육과정**: 고등학교 미적분Ⅰ
- **공식**: `PE(pos, 2i) = sin(pos / 10000^(2i/d))`, `PE(pos, 2i+1) = cos(pos / 10000^(2i/d))`

### 복소수 (Complex Numbers)

- **LLM 활용**: RoPE에서 위치 인코딩을 복소수 평면의 회전으로 해석
- **교육과정**: 고등학교 공통수학1
- **공식**: `e^(iθ) = cosθ + i·sinθ` (오일러 공식)

### 집합론 (Set Theory)

- **LLM 활용**: 어휘집(Vocabulary)이 유한 집합. Top-k, Top-p 필터링
- **교육과정**: 고등학교 대수

### 조합론 (Combinatorics)

- **LLM 활용**: 어휘 크기 V에서 길이 T 시퀀스의 가능한 조합 = Vᵀ. Beam Search 탐색 공간
- **교육과정**: 고등학교 공통수학1

## 핵심 공식 요약

### Transformer Scaled Dot-Product Attention

```
Attention(Q, K, V) = softmax(QKᵀ / √dₖ) · V
```

- **QKᵀ**: 행렬 곱 + 전치 (선형대수)
- **√dₖ**: 분산 정규화 (통계) + 차원의 저주 대응
- **softmax**: 지수 함수 + 정규화 (미적분 + 확률)
- **역전파**: 편미분 + 연쇄 법칙 (미적분)

### 학습 목적 함수 (Cross-Entropy Loss)

```
L = -Σ log P(xₜ | x₁,...,xₜ₋₁)
```

- **조건부 확률** (확률)
- **로그** (미적분 + 정보이론)
- **최대 우도 추정**과 동치 (통계)

### Layer Normalization

```
LayerNorm(x) = γ × (x - μ) / √(σ² + ε) + β
```

- **평균/분산** (통계)
- **정규화** (수치해석)

### GELU 활성화 함수

```
GELU(x) = x × Φ(x) = x × ½[1 + erf(x/√2)]
```

- **가우시안 CDF** (확률)
- **오차 함수** (해석학)

## 학습 로드맵 (교육과정 기준)

### 고등학교 수준에서 시작 가능

벡터, 내적, 삼각함수, 지수/로그 함수, 기본 미적분, 조건부 확률, 기대값/분산

### 대학교 학부 필수

1. **선형대수학** — 행렬 연산, SVD, 고유값 분해
2. **다변수 미적분학** — 편미분, 그래디언트, 연쇄 법칙
3. **확률과 통계** — MLE, 확률 분포, 베이즈 정리
4. **최적화이론** — 경사 하강법, 볼록/비볼록 최적화
5. **수치해석** — 부동소수점, 수치 안정성

### 대학원 수준 (이론 연구용)

측도론, 함수 해석학, 미분 기하학, 고급 확률과정론
